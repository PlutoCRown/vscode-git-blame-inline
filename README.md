# Git Blame Inline

轻量级的 VSCode Git Blame 插件，在代码行尾显示 blame 信息。

## 功能特性

- 🎯 在代码行尾显示 Git blame 信息（作者、时间、提交信息）
- 🔍 鼠标悬停查看完整的提交详情
- 🎨 自动适配编辑器主题（深色/浅色）
- ⚡ 高性能：智能缓存，仅处理可见区域

## 使用方法

1. 打开一个 Git 仓库中的文件
2. 自动在每行代码末尾显示灰色的 blame 信息
3. 将鼠标悬停在代码上查看完整提交详情

## 命令

- `Git Blame Inline: Toggle Inline Blame` - 切换显示/隐藏 blame 信息

## 配置

- `gitBlameInline.enabled` - 启用/禁用行内 blame 信息显示（默认：true）
